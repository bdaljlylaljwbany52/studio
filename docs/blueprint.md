# **App Name**: BizFlow

## Core Features:

- Dashboard Metrics: Display a dashboard with key metrics like revenue, expenses, and profit, all calculated from entered data.
- Appointment Calendar: Implement a simple calendar view for scheduling and displaying appointments.
- Data Entry: Allow entry of revenue and expense data, storing it in local storage or a similar client-side storage mechanism.

## Style Guidelines:

- Primary color: Neutral grays for a professional look.
- Secondary color: Soft blues for data visualization and accents.
- Accent: Teal (#008080) for interactive elements and highlights.
- Clean, card-based layout for the dashboard and calendar.
- Use clear and simple icons for navigation and data representation.
- Subtle transitions for loading data and navigating between views.
